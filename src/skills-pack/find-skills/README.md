# find-skills

Pre-installed with every ZERO AGENT. Discover and install skills from anywhere on the internet in plain English.

## Usage

```bash
zeroagent find "track crypto prices"
zeroagent find "get latest news"
zeroagent find "shorten a url"
zeroagent find "send emails"
zeroagent find "manage my calendar"
```

## Example Output

```
  🔍  Skills found for "track crypto prices"

  1. crypto-price
     Get live price of any cryptocurrency
     🟣 skills.sh
     Install: zeroagent install skills:crypto-price

  Browse more at: https://skills.sh
```

## Install Any Skill From Anywhere

ZERO AGENT is compatible with any skill from anywhere:

```bash
zeroagent install skills:skill-name
zeroagent install github:username/repo
zeroagent install npm:package-name
zeroagent install https://any-url
```
