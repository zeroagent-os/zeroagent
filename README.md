## What is ZERO AGENT?

ZERO AGENT is a foundational platform that lets you build, run, and manage AI agents ‚Äî without being locked into any proprietary system.

Every agent starts from zero. You install the skills it needs, nothing more. Skills can be added, updated, or removed at runtime ‚Äî no restarts, no vendor fees, no black boxes.

Browse and install skills from the community marketplace ‚Äî web search, crypto monitoring, social posting, data analysis and more. Or build your own and share it with the world.

Think of it as the infrastructure layer for AI agents. You own it. You control it. The community builds on top of it.
